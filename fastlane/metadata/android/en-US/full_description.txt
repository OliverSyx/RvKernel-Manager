<p>Unlock the true potential of your Android device with RvKernel Manager, a powerful and user-friendly open-source tool for managing your kernel settings. Designed for advanced users, this app provides comprehensive control over your device's kernel parameters.</p>

<p>Fine-tune your device for peak performance, maximum battery life, or a perfect balance between the two.</p>

<p><b>Features:</b></p>

<p><b>Device Information Dashboard</b><br>
Monitor critical device information including CPU model, GPU model, RAM, kernel version, and Android version details.</p>

<p><b>Battery Monitoring & Control</b><br>
Real-time battery monitoring with voltage, temperature, and charging level information. Includes thermal profile management and fast charging control for supported devices.</p>

<p><b>SoC (System on Chip) Management</b><br>
Advanced CPU frequency control with support for different core clusters (Little, Big, Prime). Adjust CPU governors and set minimum/maximum frequencies for each cluster individually.</p>

<p><b>Kernel Parameter Tuning</b></p>
<p>Fine-tune kernel settings including:</p><ul>
<li>Scheduler auto group configuration</li>
<li>Swappiness control for memory management</li>
<li>ZRAM size and compression algorithm settings</li>
<li>Kernel printk logging levels</li>
<li>Etc</li></ul>
<p>and there are many other parameters that you can tune!<p>

<p><b>Material 3 Expressive Design Interface</b><br>
Clean, modern interface with support for dynamic theming and blur effects.</p>

<p><b>Requirements:</b><br>
• ROOTED device<br>
• Android 12+<br>
• For now only suitable with Snapdragon devices</p>

<p>Take full control of your device's kernel. Tune performance, battery life, and more!</p>
